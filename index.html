Kaataq – Lenten Alutiiq Game

A digital adaptation of the traditional Alutiiq stick guessing game
Mobile-friendly. Multiplayer. Cultural education.

About the Game

Kaataq is an Alutiiq guessing game involving psychological gameplay and social interaction. Players take turns hiding a marked stick (wee) in one of their hands, while others try to guess which hand contains it. This digital version preserves the essence while adding modern multiplayer functionality.

Features

  • Real multiplayer: Cross-device gameplay using Firebase Realtime Database
  • Room-based system: Create or join games with 4-digit codes
  • Cultural education: Integrated info about Alutiiq traditions/language
  • Mobile-responsive: Optimized for phones and tablets
  • Real-time sync: All players see actions instantly
  • Authentic terminology: Uses Alutiiq words (Camiq, Taliq, unsure about it but possibly wee and dip as well)

How to Play

  1. Create or join a room using a 4-digit code
  2. Wait for players (3–8 required)
  3. Take turns as Stick Holder (choose which hand hides the "wee")
  4. Other players vote on which hand they think has the marked stick
  5. Score points for correct guesses and successful bluffing
  6. Game ends after everyone has been the Stick Holder once

Cultural Context

This game is based on Alutiiq gaming practices during the lenten season. I sourced the information from the Alutiiq Museum and Archaeological Repository to ensure authenticity and respect for the traditions.

  • Kaataq: The stick game itself
  • Wee: The marked stick
  • Dip: The unmarked stick
  • Camiq: Left hand
  • Taliq: Right hand
Note: I am not Alutiiq, I am an Orthodox Christian who has a deep appreciation for their culture.
This project is meant both as a tool for cultural exchange and to honor the spirit of Alutiiq tradition through play.
